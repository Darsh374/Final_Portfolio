What is HTML?

HTML is a Hyper Text Markup Language. HTML is the standard markup language for Web pages. The element in HTML are the building blocks of HTML pages. It is also the standard for creating web applications. With the use of JavaScript(JS) and Cascading Style Sheets(CSS). The elements are represented by <>tags.

Where can you learn?

With the internet being so accessible to people all over the world, learning can be done anywhere anytime. There are many online teachers and websites to help you get started. Some of the best websites are to learn are W3schools.com and Codecademy.com.

Why Learn HTML?

There are so many reasons why one should learn HTML with CSS and JavaScript. HTML and CSS are the basic programming languages for web development and design. This is not only beneficial for developers but marketers too. With HTML you can use it for formatting a blog or email. The advantages of learning HTML are limitless, both creativity and career wise.
